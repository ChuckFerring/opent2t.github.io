Using Onboarding Modules in Node

[ Improve this document ]

Next, let's install the onboarding module for wink hub. This is published to npm, and the source lives in the onboarding repo if you want to check it out:

$ npm install opent2t-onboarding-winkhub


Now let's run the test script included with the onboarding module for wink hub, passing in the same parameters specified in manifest.xml. Onboarding will ask for some information like the Wink API client ID, secret, and user credentials. You need to get a client ID and secret from Wink, as indicated earlier, and will need to create user credentials in the Wink app.

$ node node_modules/opent2t-onboarding-winkhub/test.js -n "Wink Light Bulb" -f "light_bulb_id"
